## Experiments

This folder contains small, focused programs used to explore and understand specific concepts before integrating them into the main application.

These experiments are not part of the core password manager and do not directly contribute to its functionality. Instead, they serve as a sandbox for testing ideas, learning new libraries, and validating approaches in isolation.

Examples of what may be found here include:
- Testing encryption methods (e.g. Fernet)
- Exploring file handling techniques
- Prototyping new features before integration
- Understanding Python concepts in a controlled environment

This approach allows for safer development by separating learning and experimentation from the main codebase.
